栈的应用举例

数制转换

十进制数n和其他d进制数的相互转换，简单算法基于这样的原理：

n = (n div d) * d + n mod d

我们计算过程是从低位到高位顺序产生，而打印输出要从高位到低位进行，和计算过程相反。所以在计算进程中各位顺序进栈，然后按出栈序列打印输出即为最后转换之后的d进制数。代码中用到了前面的示例程序：数据结构系列5栈的PHP的实现和栈的基本操作

括号匹配

程序员都很多IDE都有代码提示功能，其中一个细节就是括号匹配，实时检查代码的语法格式是否正确，并会有相应的提示，我们通过栈来实现检查括号是否为正确的格式：

((()()()())) 为正确格式

(()())) 为不正确格式
